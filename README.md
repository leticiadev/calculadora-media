# 📊 Calculadora de Média de Alunos

Este projeto é uma aplicação simples em JavaScript que calcula a média de um aluno com base em três notas e informa sua situação: **Aprovado**, **Recuperação** ou **Reprovado**.

---

## 🚀 Funcionalidades

* Inserir nome do aluno
* Inserir 3 notas
* Validação de campos vazios
* Validação de notas (0 a 10)
* Cálculo automático da média
* Exibição da situação do aluno:

  * ✅ Aprovado
  * ⚠️ Recuperação
  * ❌ Reprovado
* Feedback visual com cores (CSS)

## 📸 Como funciona

1. O usuário preenche o nome e as três notas
2. Clica no botão **"Calcular Média"**
3. O sistema:

   * Valida os dados
   * Calcula a média
   * Define a situação
   * Mostra o resultado na tela

---

## 🧠 Regras de negócio

* Média ≥ 7 → **Aprovado**
* Média entre 5 e 6.9 → **Recuperação**
* Média < 5 → **Reprovado**
